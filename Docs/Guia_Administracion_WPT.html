🛠️ Admin API (Endpoints)

Usa estos endpoints en Swagger para cambiar la configuración en caliente:

GET /api/admin/config

Ver base de datos actual de la instancia.

POST /api/admin/database

Actualizar cadenas de conexión dinámicamente.

💾 Cómo cambiar la Base de Datos de un Modelo

Envía un POST al endpoint /api/admin/database con este formato:

{
  "instanceName": "Nombre-de-la-Instancia",
  "mainConnectionString": "Server=WPT-SERVER;Database=NUEVA_BD;User Id=sa;...",
  "externalConnectionString": "Server=WPT-SERVER;Database=BD_COMERCIAL;..."
}

El sistema guardará el cambio en el appsettings.json local y reiniciará la instancia automáticamente.

🚀 Flujo de Réplica (Skill)

  1. Realiza cambios en WPTExecutor y valida.
  2. Para replicar a producción, abre PowerShell (Admin) y ejecuta:
.\Tools\Promocionar-Cambios.ps1 -Destino "eXcomercial"

Este comando sincroniza los binarios pero respeta la base de datos de producción.
